To view the source code, you need to install the Environment, NetBeans IDE 5.5 or above, and open "Devanagari Character Recognition" folder as a project.

For successful running of the Program, you need to install jdk version 1.5 or above.

You can simply now run the Project from the IDE.

This is all about source code. If you want to run the project directly , we have provided a .jar file, Devanagari_OCR.jar.
